# T-SHOP - Магазин футболок

## Технологии
- **Frontend**: Next.js 14 (App Router)
- **База данных**: PostgreSQL (Onrezа)
- **Хостинг**: Onrezа (Россия)
- **Telegram**: уведомления о заказах

## Развертывание

### 1. Регистрация на Onrezа
- Зайди на https://onreza.ru
- Зарегистрируйся

### 2. Создание базы данных
- В панели управления перейди в PostgreSQL
- Создай базу данных (Hobby бесплатно)
- Скопируй строку подключения (DATABASE_URL)

### 3. Создание Telegram бота
- Напиши @BotFather в Telegram
- Создай бота: `/newbot`
- Скопируй токен
- Узнай свой CHAT_ID: напиши @userinfobot

### 4. Деплой
- Установи CLI: `npm i -g onreza-cli` (или используй веб-интерфейс)
- Загрузи проект на GitHub
- Подключи репозиторий в Onrezа
- Добавь переменные окружения:
  - `DATABASE_URL` — строка подключения к БД
  - `TELEGRAM_BOT_TOKEN` — токен бота
  - `TELEGRAM_CHAT_ID` — твой ID

### 5. Инициализация базы
- После первого деплоя открой `/api/init` один раз для создания таблиц

## Функционал
- Каталог товаров с фильтрацией
- Админ-панель (5 кликов на логотип → пароль admin123)
- Заказы приходят в Telegram
